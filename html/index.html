Public Domain Works Archive

A (very incomplete) listing of dedications to the public domain from across the Internet.

What's Included

The primary criterion is, of course, that a work has been dedicated to the public domain. This intentionally excludes any works that were at some point copyrighted and have since aged into the public domain (in some jurisdictions), or works that are public domain for statuatory reasons. Of course, it is questionable in what, if any, jurisdictions one can sucessfully place a work in the public domain, and what means would sucessfully do so. The standard here is making an explicit effort, such as using CC0 or simply marking a work Public Domain.

The secondary criteria is whether a work is complete. For software this is a very subjective measure, but the general guideline is that a beta-quality project is sufficiently complete to include.

Haskell Serialization safecopy
  • hackage
  • website (archived)
  • github
 SafeCopy extends the parsing and serialization capabilities of Data.Serialize to include nested version control. Nested version control means that you can change the definition and binary format of a type nested deep within other types without problems.
